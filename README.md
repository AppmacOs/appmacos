## `02` В· How We Verify Every App

No app goes live without passing all 5 stages:

```
STAGE 1  в”Җв”Җ  Static analysis
             в””в”Җ entitlements, code signing, binary inspection

STAGE 2  в”Җв”Җ  Malware scan
             в””в”Җ multi-engine scan, spyware + adware detection

STAGE 3  в”Җв”Җ  Apple Notarization
             в””в”Җ must pass Gatekeeper вҖ” no exceptions

STAGE 4  в”Җв”Җ  SHA-256 checksum
             в””в”Җ generated, stored, published publicly

STAGE 5  в”Җв”Җ  Community review
             в””в”Җ 7-day open period before stable listing
```

**Check any download yourself in 10 seconds:**

```bash
shasum -a 256 ~/Downloads/your-app.dmg
# paste the output вҶ’ compare at appmacos.com/verify
```

## `05` В· Submit Your App

```bash
# 1. Fork this repo
# 2. Add your entry to /apps/
# 3. Open a PR titled: [NEW APP] AppName vX.X.X
# 4. We review within 48h
```

Your `/apps/your-app.json` must look like this:

```json
{
  "name"              : "Your App",
  "developer"         : "Your Name",
  "version"           : "1.0.0",
  "category"          : "developer-tools",
  "license"           : "free",
  "website"           : "https://yourapp.com",
  "download_url"      : "https://yourapp.com/release.dmg",
  "sha256"            : "abc123...",
  "apple_notarized"   : true,
  "universal_binary"  : true,
  "min_macos"         : "12.0",
  "description"       : "One clear sentence about what it does."
}
```

```
RULE #1  вҶ’  apple_notarized must be true. No exceptions.
RULE #2  вҶ’  sha256 must be correct. We verify it.
RULE #3  вҶ’  description must be original. No copy-paste from App Store.
```
